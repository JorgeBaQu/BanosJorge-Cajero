Precondición Estar logueado y tener saldo
Secuencia normal Paso Acción

Postcondición Que no se actualce el saldo

2 Si El numero introducido es mayor que el saldo, el sistema pone un mensaje de error por falta de saldo, a continuación este caso de uso continúa
3 Si El sistema no le resta el saldo y no se actualiza la base de datos, el sistema da error y avisa a un administrador, a continuación este caso de uso continúa
